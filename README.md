# DB Tables
I made three tables for my app, User, Tweet and Profile. User stores the user information. Tweet stores the information message that a user tweets and the Profile tab stores information about the user.

## User
  - email 
  - password

## Profile
  - avatar
  - first_name
  - last_name
  - country
  - user_id (belongs_to)

## Tweet
  - message
  - user_id (belongs_to)
